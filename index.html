My Projects

Dart Assistant Game Application [DAG]

Jan 2024 - Now

During my trip in Ireland, I discovered a massive Dart player community in pubs especially. I invited my parents to visit some places i knew and that's how my father became a dart passionnate.

Last christmas, we offered him a dart board to play with us but we didn't found a great dart game application. This is why I developped my own Dart Game Application.

This application is developped on Unity, in C# and is available on Android and Windows.

42Sh | POSIX Shell

Jan 2024 - Feb 2024

As part of a project in the third year of my engineering school, I had to create a shell in C. This shell had to be POSIX compliant and had to be able to execute commands, manage pipes, redirections, separators, variables, aliases, history, globbing, etc.

It was a 4 people project and we had to work with a git repository and a CI/CD pipeline. We learned a lot about teamwork and about planning a project with a lot of features.

This project is developped in C and is available on Linux.

Personnal Vocal Assistant

Nov 2023 - Dec 2023

During this period, I had the will to create my own vocal assistant. I wanted to create a vocal assistant that could be used on a Raspberry Pi for exemple especially to help be in my daily life.

It was also a great way to learn more about APIs and how to use them (especially the Spotify, openweather and openai APIs).

This project is developped in Python and is available on Windows and Linux.

Optical Character Recognition

Sep 2022 - Dec 2022

As part of a project in the second year of my engineering school, I had to create an OCR in C. This OCR had to be able to recognize a sudoku partially filled and to solve it.

It was a 4 people project and we had to complete this project in 3 months. I was mainly in charge of the machine learning part of the project. I successfully created a neural network that was able to recognize digits with a 89% accuracy.

This project is developped in C and is available on Linux.

Era of Guardians : Timeless Village

I was mainly in charge of the game design and the game mechanics (movements and combat system). I also created the UI and the menus of the game.

This project is developped in C# and is available on Windows. You can visit the Website.
